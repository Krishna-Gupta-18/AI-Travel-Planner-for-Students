# AI-Travel-Planner-for-Students

ğŸŒ **AI Travel Planner for Students** is an interactive web app built with **Streamlit** and **scikit-learn** that recommends travel destinations in India based on user preferences.  

Students can input their **budget, trip duration, travel type, preferred climate, and season**, and the app suggests the best destinations with **cost estimates, duration, and cost per day**.

---

## **Features**

- Select your **Budget** and **Trip Duration**  
- Choose **Travel Type**: Adventure, Cultural, Relaxation, Party  
- Choose **Preferred Climate**: Cold, Moderate, Warm, Hot  
- Choose **Preferred Season**: Summer, Winter, Monsoon  
- Get top **5 recommended destinations**  
- View **Cost per Day** and other trip metrics  
- Highlight the **cheapest destination per day**  
- Summary table for all recommendations
